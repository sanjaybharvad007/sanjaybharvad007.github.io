## How to publish (GitHub Pages)

1. Create a **public** repo named **`sanjaybharvad007.github.io`** in your GitHub account.
2. Download this folder and push it:
   ```bash
   git init
   git remote add origin https://github.com/sanjaybharvad007/sanjaybharvad007.github.io.git
   git add .
   git commit -m "Initial portfolio site"
   git branch -M main
   git push -u origin main
   ```
3. Pages will auto-deploy. Your site will be available at:  
   **https://sanjaybharvad007.github.io**
